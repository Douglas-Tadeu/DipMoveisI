**A ideia inicial era desenvolver um app em que fosse possivel cadastrar restaurantes e pratos seguindo algumas regras.**

As regras eram: 
- Os restaurantes poderiam cadastrar seus próprios pratos.
- O dono do restuarante não poderia avaliar seus próprios pratos.
- Um usuário não poderia avaliar o mesmo prato mais de uma vez.
- O sistema deveria calcular a média de avaliações.

Devido ao pouco tempo e às muitas atividades de outras disciplinas em paralelo (Socorro Deus! Pra quem é ateu, não sei a quem pedir ajudar!), 
o meu projeto atual foi simplificado para cadastrar meus pratos favoritos, contendo: 
- Nome do prato
- Descrição
- Valor
- URL do prato
Após cadastrar o prato, eu tenho a opção de avaliá-lo.
O App roda em memoria, sendo assim, caso feche a aba ou abra outra, as informações irão sumir. Esse problema eu deixo para versão 2.0.

- **Git & GitHub**

## Como Executar o Projeto

### Clonar o repositório

### Acessar o diretório
- Em um terminal, você utiliza para o servidor: food_travel_server
- Comando: node server.js
- Em outro terminal você vair rodar o projeto com os comandos a seguir. 

### Instalar dependências
- Comando: flutter pub get
  
### Executar o app
- Comando: flutter run

**Como utilizei o chrome para rodar, o comando foi: flutter run -d chrome**
